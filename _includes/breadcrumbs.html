{%- comment -%} Breadcrumbs include Usage: included automatically from default layout. Builds crumbs from page.url and attempts to resolve titles by matching site.pages and site.notes by URL. Fallback: humanize URL segments if no page found. {%- endcomment -%} {%- assign url = page.url | default: '/' -%} {%- assign parts = url | split: '/' | where_exp: 'seg', 'seg != blank' -%}
